# SQL

## DDL
> Data Definition Language
- `CREATE`
  - CREATE DATABASE
  - CREATE TABLE
- `ALTER`
  - ALTER TABLE
- `DROP`
  - DROP DATABASE
  - DROP TABLE


## DML
> Data Manipulation Language
- `INSERT`
- `UPDATE`
- `DELETE`
- `TRUNCATE`


## DTL
> Data Transaction Language
- `COMMIT`
- `BEGIN`
- `ROLLBACK`
  

## DQL
> Data Query Language
- `SELECT`


## DCL
> Data Control Language
- `GRANT`
- `REVOKE`
